= Best Practices (Security)
:page-toc: right
:toclevels: 2
:description: WAF++ Security – Overview of 7 security best practices with concrete implementation guidance and Terraform examples.

The following best practices translate the WAF-SEC controls into concrete,
actionable implementation guidance. Each best practice contains:

* Context and problem description
* Technical implementation steps
* Terraform examples (compliant and non-compliant)
* Anti-patterns
* Metrics for measuring success

== Where to Start?

For organizations that are new to cloud security:

. *Immediately (Day 1):* xref:best-practices/bp-iam.adoc[IAM Best Practice] – Without a solid IAM foundation, everything else is useless
. *This week:* xref:best-practices/bp-secrets.adoc[Secrets Management] – Hardcoded credentials are an immediate risk
. *This month:* xref:best-practices/bp-network-security.adoc[Network Security] – Set up network segmentation
. *This quarter:* xref:best-practices/bp-encryption.adoc[Encryption Strategy] – Introduce CMK for sensitive data
. *Longer term:* xref:best-practices/bp-zero-trust.adoc[Zero Trust] – Maturity-level-dependent transformation
