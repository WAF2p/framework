= Begriffe
:description: Glossar wichtiger Begriffe im WAF++ Framework

== Zentrale Begriffe

*Säule*::
Eine der sieben fundamentalen Dimensionen des WAF++ Frameworks (Security, Cost Optimization, Performance Efficiency, Reliability, Operational Excellence, Sustainability, Compliance/Governance/Data Sovereignty).

*Reifegrad / Reifegradmodell*::
Maturity / Maturity Model – Ein Bewertungsrahmen zur strukturierten Einschätzung des aktuellen Entwicklungsstands einer Cloud-Plattform.

*Governance*::
Governance (englischer Begriff wird beibehalten) – Strukturierte Steuerung und Kontrolle von Prozessen, Richtlinien und Entscheidungen.

*Cloud-agnostisch*::
Cloud-agnostic – Unabhängig von einem spezifischen Cloud-Anbieter; auf mehrere Provider übertragbar.

*Souverän / Souveränität*::
Sovereign / Sovereignty – Kontrolle über eigene Daten, Infrastruktur und Entscheidungen; Unabhängigkeit von externen Anbietern.

*TSC*::
Technical Steering Committee – Gremium zur technischen Steuerung des Projekts.

*RFC*::
Request for Comments – Formeller Vorschlag zur Diskussion neuer Features oder Änderungen.
