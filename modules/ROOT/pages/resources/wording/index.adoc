= Terminology
:description: Glossary of key terms in the WAF++ framework

== Core Terms

*Pillar*::
One of the seven fundamental dimensions of the WAF++ framework (Security, Cost Optimization, Performance Efficiency, Reliability, Operational Excellence, Sustainability, Compliance/Governance/Data Sovereignty).

*Maturity / Maturity Model*::
A scoring framework for structured assessment of the current state of development of a cloud platform.

*Governance*::
Structured management and control of processes, policies, and decisions.

*Cloud-agnostic*::
Independent of a specific cloud provider; transferable to multiple providers.

*Sovereign / Sovereignty*::
Control over one's own data, infrastructure, and decisions; independence from external vendors.

*TSC*::
Technical Steering Committee – the body responsible for technical governance of the project.

*RFC*::
Request for Comments – a formal proposal for discussion of new features or changes.
