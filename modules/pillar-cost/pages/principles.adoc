= Cost Optimization Principles

== P1 – Transparency First
Alle Cloud-Kosten müssen nachvollziehbar und zuordenbar sein.

== P2 – Ownership
Jede Ressource hat einen klaren Owner.

== P3 – Automation
Budget-Kontrollen erfolgen automatisiert.

== P4 – Continuous Optimization
Kosten werden regelmäßig überprüft und optimiert.
