= Sovereignty Principles

== P1 – Data Location Transparency
Speicherorte müssen dokumentiert und überprüfbar sein.

== P2 – Jurisdiction Awareness
Rechtliche Einflüsse auf Datenverarbeitung müssen bewertet werden.

== P3 – Exit Strategy
Ein Anbieterwechsel muss technisch möglich sein.

== P4 – Encryption by Default
Daten sind standardmäßig verschlüsselt.
