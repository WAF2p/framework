= Governance (Säule 7)
:page-navtitle: Governance
:page-toc: right
:description: WAF++ Säule 7 – Governance als strukturierende und auditierbare Disziplin innerhalb des Frameworks.

Governance ist eine von sieben gleichwertigen Säulen des WAF++.

Sie definiert, wie Anforderungen aus allen anderen Säulen:

* versioniert
* überprüfbar
* erzwingbar
* und auditierbar

gemacht werden.

Governance ersetzt keine andere Säule.
Sie strukturiert deren Umsetzung.

== Rolle innerhalb der 7 Säulen

* Security definiert Schutzmaßnahmen.
* Reliability definiert Resilienz.
* Cost definiert Wirtschaftlichkeit.
* Performance definiert Effizienz.
* Sustainability definiert ökologische Verantwortung.
* Sovereign definiert Souveränität.
* Governance definiert, wie diese Anforderungen technisch kontrolliert werden.

Governance ist die Operationalisierung von Architekturqualität.
