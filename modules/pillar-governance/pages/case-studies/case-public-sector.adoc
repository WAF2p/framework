= Case Study – Public Sector Data Sovereignty
:page-toc: right

== Ausgangssituation

Behörde mit regulatorischer Bindung an nationale Datenresidenz.

Herausforderungen:
* Multi-Region Default Settings
* Unklare Backup-Locations
* Unzureichende Break-Glass Kontrolle

== Maßnahmen

* Region Pinning Enforcement
* Logging Residency Fix
* Break-Glass mit Hardware MFA
* Exit-Drill durchgeführt

== Betroffene Controls

* WAF-SOV-010
* WAF-SOV-020
* WAF-SOV-040
* WAF-SOV-070
* WAF-SOV-100

== Ergebnis

* 0 Region Violations
* Vollständige Auditierbarkeit
* Zertifizierungsfähigkeit erreicht
