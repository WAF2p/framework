= Maturity Model

WAF++ definiert Reifegrade basierend auf
aggregierten Control-Ergebnissen.

== Level 1 – Initial
Einzelne Controls erfüllt.

== Level 2 – Managed
Mehrheit der Controls aktiv umgesetzt.

== Level 3 – Defined
Alle mandatory Controls erfüllt.

== Level 4 – Optimized
Kontinuierliche Verbesserung nachweisbar.
