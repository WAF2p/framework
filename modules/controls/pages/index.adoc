= WAF++ Controls (Maschinenlesbare YAML-Controls)
:page-navtitle: Controls Overview
:page-toc: right

Dieses Modul enthält alle maschinenlesbaren Controls des WAF++ Frameworks als YAML-Dateien.
Die Controls können direkt vom WAF++ Checker Tool gegen Terraform-Code ausgeführt werden.

|7 · Sovereign
|`WAF-SOV-xxx`
|xref:pillar-sovereign:controls.adoc[Controls (Sovereign)]

|===

* Eine YAML-Datei pro Control unter `modules/controls/controls/`
* IDs folgen dem Schema `WAF-{PILLAR}-{NNN}` (z.B. `WAF-COST-010`, `WAF-SOV-020`)
* Den vollständigen Katalog mit Details findest du unter xref:catalog.adoc[Controls Catalog]
