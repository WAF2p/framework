= Sovereign Controls (Control-Modul)
:page-navtitle: Sovereign Controls
:page-toc: right

Dieses Modul enthält die maschinenlesbaren Controls für die WAF++ Säule 7 *Sovereign* als YAML.

* IDs folgen dem Schema `WAF-SOV-xxx`
* Eine Datei pro Control: `modules/controls-sovereign/controls/WAF-SOV-010.yml` etc.
* Die Texte / Intents befinden sich im Säulenmodul: xref:sovereign.adoc[Controls (Sovereign)]
