= WAF++ Controls (Machine-Readable YAML Controls)
:page-navtitle: Controls Overview
:page-toc: right

This module contains all machine-readable controls of the WAF++ Framework as YAML files.
The controls can be executed directly by the WAF++ Checker Tool against Terraform code.

[cols="1,1,2"]
|===
|Pillar |Prefix |Narrative

|1 · Security
|`WAF-SEC-xxx`
|xref:pillar-security:controls.adoc[Controls (Security)]

|2 · Cost Optimization
|`WAF-COST-xxx`
|xref:pillar-cost:controls.adoc[Controls (Cost Optimization)]

|7 · Sovereign
|`WAF-SOV-xxx`
|xref:pillar-sovereign:controls.adoc[Controls (Sovereign)]

|===

* One YAML file per control under `modules/controls/controls/`
* IDs follow the schema `+WAF-{PILLAR}-{NNN}+` (e.g. `WAF-COST-010`, `WAF-SOV-020`)
* The complete catalog with details can be found at xref:catalog.adoc[Controls Catalog]
