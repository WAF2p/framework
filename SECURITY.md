# SECURITY.md

## Security Policy

Die Sicherheit von WAF++ und seiner Nutzer hat höchste Priorität.

### Meldung von Sicherheitslücken

**Bitte keine Security-Issues öffentlich erstellen.**

Stattdessen:

* E-Mail an **[security@waf2p.org](mailto:security@waf2p.org)**
* Beschreibe das Problem so detailliert wie möglich
* Falls vorhanden: Proof-of-Concept oder Reproduktionsschritte

### Was passiert danach?

* Wir bestätigen den Eingang innerhalb weniger Tage
* Wir bewerten die Auswirkung und stimmen ggf. einen Fix ab
* Responsible Disclosure ist ausdrücklich erwünscht
